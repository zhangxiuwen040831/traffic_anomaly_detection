# 网络流量异常检测系统 - 项目概览

## 1. 项目简介

本项目实现了一个基于深度学习的网络流量异常检测系统，采用 Autoencoder 架构进行无监督异常检测。系统能够自动识别偏离正常行为模式的网络流量，为网络安全提供新的保障手段。

## 2. 研究背景

随着网络攻击手段的不断演进，传统的基于规则的安全防护方法已经难以应对复杂的网络威胁。据统计，2025年全球网络攻击事件增长率超过40%，传统方法的检测率不足70%。

异常检测作为一种基于机器学习的方法，能够通过学习正常网络流量的模式，自动识别异常行为，为网络安全防护提供了新的思路。

## 3. 项目目标

- 开发基于深度学习的网络流量异常检测系统
- 实现端到端的模型训练、评估和部署流程
- 构建用户友好的前端界面
- 验证系统在真实数据集上的性能

## 4. 技术栈

- **后端**：Python, PyTorch, scikit-learn
- **前端**：Streamlit
- **部署**：Docker, REST API
- **数据处理**：Pandas, NumPy

## 5. 系统架构

### 5.1 总体架构
- **数据层**：数据采集、预处理、特征工程
- **模型层**：Autoencoder 系列模型，混合异常评分
- **评估层**：多维度指标评估，阈值优化
- **部署层**：模型打包，推理服务
- **前端层**：Streamlit 交互式界面

### 5.2 核心组件
- **数据预处理**：特征标准化、编码、缺失值处理
- **模型训练**：仅使用正常样本进行无监督学习
- **异常评分**：重构误差、潜在空间距离、密度估计的混合评分
- **阈值优化**：基于验证集的 f1_optimal、youden、pr_optimal 策略
- **推理服务**：批处理和单样本推理
- **前端界面**：数据上传、实时推理、结果可视化

## 6. 核心技术

### 6.1 模型选择
- **增强型 MLP Autoencoder**：添加批归一化和 Dropout，提高模型泛化能力
- **VAE**：变分自编码器，捕获数据分布
- **Hybrid AE**：结合多种评分组件

### 6.2 异常评分方法
- **重构误差**：衡量输入与重构输出的差异
- **潜在空间距离**：计算样本在潜在空间中与正常样本分布的距离
- **密度估计**：基于正常样本的密度分布评估异常程度
- **混合评分**：加权组合三种评分方法

### 6.3 阈值策略
- **F1 最优阈值**：最大化 F1 分数
- **Youden 指数**：平衡灵敏度和特异度
- **PR 最优阈值**：最大化精确率和召回率的乘积

### 7.2 实用价值

## 9. 最终结论

本项目成功实现了基于深度学习的网络流量异常检测系统，在真实数据集上取得了良好的检测性能。系统具有较高的精确率和可接受的召回率，适合实际部署使用。

通过无监督学习方法，系统能够在不需要标注异常样本的情况下，有效地检测网络流量中的异常行为，为网络安全防护提供了新的技术方案。
